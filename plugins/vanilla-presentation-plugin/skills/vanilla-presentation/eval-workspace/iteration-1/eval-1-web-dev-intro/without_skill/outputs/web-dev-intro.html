Section 01

HTML · CSS · JavaScript 기초

🏗️

HTML

웹 페이지의 구조를 정의합니다. 태그(tag)를 사용해 제목, 단락, 이미지, 링크 등 콘텐츠를 마크업합니다.

<h1>Hello</h1>
<p>웹 개발 시작!</p>
<a href="/">홈</a>
🎨

CSS

웹 페이지의 스타일을 담당합니다. 색상, 폰트, 레이아웃, 애니메이션 등 시각적 표현을 제어합니다.

h1 {
  color: #4f8ef7;
  font-size: 2rem;
}
⚡

JavaScript

웹 페이지에 동작을 부여합니다. 사용자 입력 처리, 데이터 조작, 서버 통신 등을 구현합니다.

const msg = '안녕!';
alert(msg);
// 브라우저에 알림 표시
HTML = 뼈대  |  CSS = 옷(스타일)  |  JavaScript = 근육(동작)
Section 02

DOM 조작

DOM(Document Object Model)은 브라우저가 HTML을 트리 구조로 표현한 객체 모델입니다. JavaScript로 DOM을 읽고 수정하면 페이지가 동적으로 변합니다.

document
└── <html>
├── <head>
│   └── <title> "웹"
└── <body>
    ├── <h1 id="title">
    └── <p class="desc">

주요 DOM API

// 요소 선택
const el = document.getElementById('title');
const all = document.querySelectorAll('.item');

// 내용 변경
el.textContent = '새 제목';
el.innerHTML = '<b>강조</b>';

// 스타일 / 클래스 조작
el.style.color = 'blue';
el.classList.add('active');

// 요소 생성 & 추가
const p = document.createElement('p');
p.textContent = '새 단락';
document.body.appendChild(p);
Section 03

이벤트 처리

이벤트는 브라우저에서 발생하는 사용자 행동이나 시스템 알림입니다. addEventListener로 원하는 이벤트에 반응할 수 있습니다.

  • 🖱
    마우스 — click, dblclick, mouseover, mouseout
  • ⌨
    키보드 — keydown, keyup, keypress
  • 📝
    폼 — submit, change, input, focus, blur
  • 🌐
    윈도우 — load, resize, scroll, DOMContentLoaded
const btn = document.querySelector('#btn');

// 클릭 이벤트 등록
btn.addEventListener('click', (e) => {
  console.log('버튼 클릭!', e.target);
  btn.textContent = '클릭됨 ✓';
});

// 키보드 이벤트
document.addEventListener('keydown', (e) => {
  if (e.key === 'Enter') {
    console.log('엔터 키 입력');
  }
});

// 이벤트 버블링 막기
el.addEventListener('click', (e) => {
  e.stopPropagation();
  e.preventDefault(); // 기본 동작 취소
});
Section 04

비동기 프로그래밍

JavaScript는 싱글 스레드이지만, 비동기 처리로 서버 요청·파일 읽기 등 오래 걸리는 작업을 블로킹 없이 처리합니다.

Callback

함수에 함수를 전달. 중첩이 깊어지면 "콜백 지옥" 발생.

Promise

비동기 결과를 객체로 표현. .then().catch() 체이닝 가능.

Async / Await

Promise를 동기 코드처럼 읽기 쉽게 작성. 현재 표준.

// ① Callback
setTimeout(() => console.log('1초 후'), 1000);

// ② Promise
fetch('/api/data')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

// ③ Async / Await (권장)
async function loadData() {
  try {
    const res  = await fetch('/api/data');
    const data = await res.json();
    console.log(data);
  } catch (err) {
    console.error('에러:', err);
  }
}
loadData();
